# Changelog

Todos los cambios notables a este proyecto serán documentados en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Agregado
- Lab 01: RDS Fundamentals - Configuración completa de Amazon RDS con SQL Server
- Lab 02: DMS Migration - Migración de datos usando AWS Database Migration Service
- Lab 03: Serverless Architecture - API REST con Lambda, DynamoDB y API Gateway
- Lab 04: Athena Quest - Análisis de datos con Amazon S3 y Athena
- Documentación completa para todos los laboratorios
- Screenshots organizados por pasos para cada lab
- Diagramas de arquitectura para cada solución
- Scripts de CloudShell y SQL para automatización
- Reportes en PDF con análisis detallado
